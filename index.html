ℹ️ About

        

          

            

              
🤖 Robotics Classroom Program

              
Version 1.0 — Browser-based, no installation needed.

            

            

              
📚 Supported Robots

              
Dobot Magician, Dobot AI Starter, Dobot Magician AI Kit, VEX

            

            

              
💻 Coding Modes

              
Blockly Visual Blocks, Python Editor, Event Sheet

            

            

              
🔗 GitHub Integration

              
Direct GitHub API — push & pull without admin permissions

            

          


          


          

            
🛠️ Dobot Python Wrapper

            
To run generated code on a real Dobot, install the wrapper:

            
pip install dobot-magician-wrapper

            

              Then run your generated .py file from the Command Prompt or Terminal.
            

          


          

            
🌐 Running Without Internet

            
The GitHub token is stored locally in your browser. GitHub API requires internet.
            Blockly and Monaco Editor load from CDN. For fully offline use, download the library files and update the script tags.
